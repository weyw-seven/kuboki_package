The meaning of each world file in thie [folder](./turtlebot_gazebo/worlds) is:  
- **my_simple_closure.world**: a square simple closure with walls and trashbins  
  ![w1](../data/world_snapshot/my_simple_closure.png)

- **my_simple_closure2.world**: a square simple closure, with clear visual feature walls  
  ![w2](../data/world_snapshot/my_simple_closure2.png)

- **my_small_maze.world**: some rooms to test laser, navigation and exploration. From explore-lite
  ![w3](../data/world_snapshot/my_small_maze.png)

- **willowgarage.world**: official gazebo world, has many rooms and is pretty large
  ![w4](../data/world_snapshot/willowgarage.png)

- **lower_obstacle.world**: a world that has many small objects that below laserscan height. It use to test the ability of obstacle avoidance.
  ![w5](../data/world_snapshot/lower_obstacle.png)

- **lower_obstacle1.world**: like the base version, but the obstacle density is more reasonable.
  ![w6](../data/world_snapshot/lower_obstacle1.png)


## Deprecated world
- **empty.world**, **corridor.world** and **playground.world** are the original three worlds.
- **my_kitchen.world**: a kitchen house, laser will go through the wall!  
- **my_cafe.world**: a simple structure cafe, useless  
- **my_simple_closure1.world**: a square simple closure with another wall
